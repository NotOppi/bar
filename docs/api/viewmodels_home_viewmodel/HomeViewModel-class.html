HomeViewModel

HomeViewModel class

ViewModel que gestiona el estado de la pantalla Home.

Implementa ChangeNotifier para notificar a la UI cuando hay cambios en la lista de pedidos. Mantiene una lista de pedidos y proporciona métodos para añadir nuevos pedidos.

Se inicializa con pedidos de ejemplo para demostración.
